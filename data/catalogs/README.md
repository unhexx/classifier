# Справочники типовых неисправностей

JSON-файлы в этой директории — **источник истины** для справочников.

## Формат

```json
{
  "name": "servers",
  "version": "2.0",
  "description": "Описание справочника",
  "faults": [
    {
      "code": "SRV-DISK-001",
      "title": "Заголовок",
      "description": "Подробное описание",
      "symptoms": ["симптом 1", "симптом 2"],
      "keywords": ["ключевое", "слово"],
      "category": "Категория",
      "recommended_actions": ["Действие 1", "Действие 2"]
    }
  ]
}
```

## Доступные справочники

| Файл | Имя | Записей | Описание |
|------|-----|---------|----------|
| `servers.json` | servers | 28 | Серверное и IT-оборудование |
| `network.json` | network | 25 | Сетевое оборудование |
| `automotive.json` | automotive | 14 | Автомобильные неисправности |
| `industrial.json` | industrial | 14 | ИБП, HVAC, хранение, ДГУ |

## Загрузка

```bash
make seed
# или
python scripts/seed.py
```

При старте сервиса сиды загружаются автоматически (идемпотентный upsert по `code`).

## Добавление нового справочника

1. Создать `<name>.json` по схеме выше
2. Запустить `make seed` или перезапустить сервис
3. Проверить: `curl http://localhost:8123/api/v1/catalogs`
